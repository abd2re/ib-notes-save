# Mole
?
Une mole est un ensemble de $N_{A}$ entités identiques.
$N_A$ représente le nombre d'Avogardo =:: $6.02 \cdot 10^{23}$

Formule nombre de moles=
?
$$n=\frac{N}{N_{A}}$$
$n$ = nombre de moles
$N$ = nombre de molécules
$N_{A}$ = nombre d'Avogardo
